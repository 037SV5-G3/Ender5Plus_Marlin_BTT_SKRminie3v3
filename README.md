# Ender5Plus_Marlin_BTT_SKRminie3v3
Marlin modificado para la SKR mini E3 V3
Esta modifiacion se realizo para la impresora ENDER 5 PLUS
Se cambió la pantalla por la stock de las ender 3
Se cambio la placa por la placa Bigtreetech SKR mini E3 V3, la cual tiene drivers silenciosos, control de corriente por UART,
sensorless homing, compatibilidad con pantallas TFT, etc.
Se reemplazó el extrusor por el kit Sprite Extruder SE compatible con la Ender 3 de Creality, para extrusion directa.
Conservando todas las demas partes, como hotend, motores paso, fuente de alimentacion, etc.
